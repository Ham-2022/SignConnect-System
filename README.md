SignConnect – ISL Based Communication System
Overview

SignConnect is an AI-powered communication platform designed to bridge the gap between Deaf and Hard-of-Hearing (DHH) individuals and hearing users. The system enables real-time bidirectional communication by translating Indian Sign Language (ISL) gestures into text/speech and converting spoken language into visual sign representations.

The platform uses computer vision, speech processing, and web-based APIs to create an accessible communication environment, especially in classrooms, public services, and daily conversations.

Key Features

✋ Real-time ISL Gesture Recognition using computer vision

🎥 Camera-based Hand Tracking using MediaPipe

🗣 Speech-to-Text Conversion for hearing users

🔊 Text-to-Speech Output for natural communication

🤟 Sign Visualization using avatars or animations

💬 Real-time Conversation Log with timestamps

⚙ System Status Indicators (camera, mic, connectivity)

🧠 Rule-based gesture recognition engine
